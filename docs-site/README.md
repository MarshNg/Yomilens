# YomiLens Documentation

Source for the YomiLens user guide published with GitHub Pages at
https://marshng.github.io/Yomilens/.

## Prerequisites

- Node.js `>=22.13.0`

## Local development

```bash
npm install
npm run dev
npm run build
```

Edit the guide in `app/page.tsx`, styling in `app/globals.css`, and screenshots
in `public/screenshots/`. Pushing changes to `main` automatically republishes
the site through GitHub Actions.
